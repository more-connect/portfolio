園について

めいぷろ幼稚園は、自然の中で身体を動かす楽しさを学び、
健康な身体づくりに取り組んでいます。

園の特徴

めいぷろ幼稚園は、職員みんなが子供一人一人をよく知り、 保護者との会話も多い、アットホームな場所です。

  • 植物や生き物とふれあい、命の大切さを学んでいきます。
  • 「遊び」を大切にし、芝生で走り回り、転げまわり、泥まみれになったりして、五感で感じ、想像力を膨らませていきます。
  • 外国人教師による異文化体験を行い、遊びながら英語に親しみ、国際感覚を育みます。
  • 防犯カメラを設置し、消毒や換気などを徹底に行い、安全快適な環境を提供します。
