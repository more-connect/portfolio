当院について

About

患者様の思いにしっかり向き合う 歯科医院です。

当院では、
患者様一人一人の症状にあった最適な処置を施します。
患者様とのコミュニケーションを大切に、
「しっかりとしたカウンセリング」を行い
予防を中心に、「未来」を見据えた治療を行います。

診療内容

Menu

一般歯科

一般的な虫歯、入れ歯の治療を行い、
お口の病気全般について診断できます。

矯正歯科

歯並びは、口元の見た目だけでなく、
健康にも影響します。

歯周病治療

歯を失う原因の一位は歯周病です。
歯茎の出血や腫れなどの異変があれば
すぐにご相談ください。

口腔外科

舌や頬の粘膜、顎などの処置を行います。
親知らずの抜歯、口内炎、
顎関節症などがあります。

ホワイトイング

歯を削らず、薬剤で白くするので、
手軽に、安全に
白い歯を手に入れることができます。

院長のあいさつ

Message

院長の田中太郎と申します。

当院では、
患者様一人一人に対話を重視したカウンセリングを行い 「未来」を見据え、患者様の立場になることを第一に考え 「丁寧」な治療を心がけています。

そのほか、最新の空気清浄システムを導入し、ウイルス、微細なホコリ、花粉などを除去し、快適な院内空間作りをしています。

治療方法や症状で気になることがございましたら、まずはお電話でお気軽にご相談ください。 何でも相談できる「かかりつけ医」となれるように、尽力してまいります。
皆様の笑顔が素敵になりますように。
