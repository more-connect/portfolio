コーディング

HTML/CSSを用いてWebサイトの構築を行います。
レスポンシブにも対応可能です。

BEM記法を意識し、保守性・拡張性の高いコーディングを行い、
運用の負担を軽減いたします。

また、CSSアニメーションやJavaScriptを用いて
Webサイトに動きを付けることも可能です。

ホームページ修正

文章・画像の変更、表示崩れの修正、情報の更新など
既存のホームページで変更を加えたいというお困りごとに対応させていただきます。
お問い合わせフォームの構築も行います。

WordPress

WordPressを用いてWebサイトの構築を行います。

また、テーマのカスタマイズや、プラグインの追加なども行います。
