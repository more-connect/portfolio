コーディング

HTML/CSSを用いてWebサイトの構築を行います。
レスポンシブにも対応可能です。

BEM記法を意識し、保守性・拡張性の高いコーディングを行い、
運用の負担を軽減いたします。

また、CSSアニメーションやJavaScriptを用いて
Webサイトに動きを付けることも可能です。

CMS

CMSとは、web制作の知識がない方でも
記事の更新や、画像の登録・更新などが簡単にできるシステムです。
その中でも、シェアトップを誇るWordPressを使用し、
お客様の日々のサイト更新の負担を減らすことができます。

オリジナルテーマ作成および、
静的サイトのWordPress化にも対応可能です。
更新性の高いサイトを作成し、運用しやすいサイトを構築します。
